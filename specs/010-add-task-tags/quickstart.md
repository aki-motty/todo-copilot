# Quickstart: Using Task Tags

## Adding a Tag

1. Open the Todo list.
2. Click on a Todo item to expand it.
3. Click the "Add Tag" button (or "+").
4. Select a tag from the dropdown list (e.g., "Research").
5. The tag will appear on the Todo item.

## Removing a Tag

1. Locate the tag on the Todo item.
2. Click the "x" icon on the tag chip.
3. The tag will be removed.
