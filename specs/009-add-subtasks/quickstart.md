# Quickstart: Add Subtasks

## Prerequisites
- Existing Todo Copilot environment setup.

## Running the Feature
1. Start the backend:
   ```bash
   npm run dev:lambda
   ```
2. Start the frontend:
   ```bash
   npm run dev
   ```

## Testing
- Run unit tests:
  ```bash
  npm test
  ```
- Run E2E tests:
  ```bash
  npx playwright test
  ```
